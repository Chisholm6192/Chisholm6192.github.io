Laurier Computing Society
Mobile App

This is a mobile app developed by myself and 2 other students and members of the Laurier Computing Society. The application was developed using React Native, and will be available on both the Google Play Store and App Store in a few weeks. It features a user authentication login system where users can create an account and have their credentials saved in our database, as well as chat with other users using our chatroom / personal messaging feature. The app also features a live youtube and twitch API to allow students watch any events hosted by the Laurier Computing Society live through their mobile devices. Finally it features a live news feed where moderators can upload news articles and content from the webpage. The backend is built using a firebase real time database system which allows for data to be uploaded and removed from the database in seconds, with live refresh to the app there are real-time updates that allow for the app to show current and relevant data.

- Development and implementation of database
- Development of clickable image and clickable link components
- Development of resources page and backend webpage for data uploads

Climate Change Analysis - Data Analytics

This is a project built for to showcase my knowledge of data science and analytics. Designed by myself and a few others from my Data Science class. This project uses data from Wilfrid Laurier Universities library databases for student usage. The data consists of excel spreadsheets with data from 1980 onwards about bio fuel production, petro liquid production, electricity, natural resource usage and natural gas production and consumption, as well as data about the quality of oxygen around the world and the state of the icecaps (data about sizes and melting rates over the years). Initially working with very untidy and very sofisticated data, one of the main pieces was to clean it up so it was computable and readable. Once the data was organized and empty/missing and useless data was removed, I was able to plot the trends of ice caps levels in comparison to worldwide production of bio fuels, petro liquids and overall polution levels over the years. The data was loaded into RStudio (an IDE for R code), and written in R, using the tidyverse library for data tidying and for graphing purposes. The final analysis/conclusion from the project was that pollution levels do indeed lead to increases in temperature and changes in climate globally.



Contributions:

- Data plotting and graphing/computing data
- Analysis of data and creation of hypothesis

2048 Game

This is a webapp remake of the very popular mobile phone game 2048. The objective of the game is for the platyer to combine different multiples of 2 repeatedly until the number 2048 is reached, without running out of space on the board. This game is built in Java using IntelliJ Idea. This game uses many GUI components of the Java language such as JFrames and the Java swing and awt libraries. There is a lot of usage of object oriented programming principles in the development of this game, as tile objects were created to be deployed upon request from the system. The game engine is built to be intelligent enough to know when a new tile must be created, and pickup the values of each time for combination.
